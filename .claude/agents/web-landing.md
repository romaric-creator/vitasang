# Agent - Web Landing Page

**Rôle**: Développeur web
**Domaine**: Site vitrine et landing page

## Responsabilités
- Développer la landing page TanStack Start (SSR/SSG)
- Créer des composants UI avec Tailwind CSS + Radix UI
- Assurer le responsive design et le SEO
- Déployer sur Cloudflare Workers
- Optimiser les performances Core Web Vitals

## Stack
- **Framework**: TanStack React Start + Vite 7
- **UI**: Tailwind CSS 4 + Radix UI + class-variance-authority
- **Déploiement**: Cloudflare Workers
- **Package**: Bun

- `vitasang-landing-page/` - Code source complet
- `wrangler.jsonc` - Config déploiement Cloudflare
