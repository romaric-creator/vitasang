EXEMPLES DE FLUX RÉELS
📢 CAS 1: Create Alert
1️⃣ POST /alerts + JWT
2️⃣ JWT verify (Middleware)
3️⃣ AlertController.create()
4️⃣ AlertService query
5️⃣ MySQL INSERT
6️⃣ Redis invalidate
7️⃣ BullMQ queue job
8️⃣ Async: Expo Push
👤 CAS 2: Get Profile
1️⃣ GET /profile/:id
2️⃣ Check Redis cache
3️⃣ 🟢 CACHE HIT (<1ms)
4️⃣ OR 🔴 MISS →MySQL (50ms)
5️⃣ Sequelize query
6️⃣ Redis SET (TTL: 1h)
7️⃣ Response + ETag
8️⃣ AsyncStorage (Local)
📅 CAS 3: Book Slot
1️⃣ POST /appointments
2️⃣ Validate slot_id
3️⃣ MySQL INSERT + UPDATE
4️⃣ Check availability
5️⃣ Redis invalidate list
6️⃣ BullMQ job: reminder
7️⃣ Scheduled for -2h
8️⃣ Expo notification
⚠️ CAS 4: Error Flow
1️⃣ Exception thrown
2️⃣ Error middleware
3️⃣ Sentry.captureException()
4️⃣ Log errorId + stack
5️⃣ Slack alert (async)
6️⃣ DB audit log
7️⃣ Response: {errorId}
8️⃣ Client retry logic
⚡ AVEC CACHE REDIS
Get Profile: <1ms | Hit Rate: 85% | TTL: 1h profile, 30m appts, 5m alerts
🔄 SANS CACHE
Get Profile: 50-100ms | MySQL query + Sequelize overhead | Network latency
